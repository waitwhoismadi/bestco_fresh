Workwise - The Freelancer Social Networking Script is a platform where Workwise will be a subtle and smart choice for user post project and receive competitive bids from freelancers within minutes. Our reputation system will make it easy to find the perfect freelancer for your job. It’s the simplest and safest way to get work done online.

Features :-
- Built with Laravel 7
- Subscription system
- Users can set the monthly subscription price
- Bootstrap 4
- Font Awesome 5
- Storage (Local)
- Notifications Real-time
- Online / offline user
- Edit your page (profile)
- Create unlimited job and project Posts
- Creators can change their name, username and password
- Creators manage their posts
- Bids from recent projects of all creators
- Apply job from recent jobs of all creators
- Infinite scroll
- Categories
- SEO
- Upload Avatar
- Upload Cover
- Search creators by Name/Username and category
- Email Verification
- Users can like in posts
- Users can comment on posts
- Ajax Comments
- Upload Portfolio
- SSL certificate(required)
- SMTP Support
- Share your page URL (Profile) or share on Facebook and Twitter
- Option Delete Account
- Message system with creators, they can use only text
- Infinite scroll on Message system
- Creator can edit posts and replace text
- Email notifications (New subscriber)
- About us page


Requirements:-
- PHP 7.2.5 or Greater
- MySQL 5.6
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension
- BCMath PHP Extension
- cURL
- GD Library or ImageMagick or Imagick.
- Fileinfo PHP Extension
